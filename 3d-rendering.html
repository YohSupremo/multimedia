Production Breakdown

01

Scene Setup & Reference Gathering

Opened Blender 5.1 and configured a new scene with a black HDRI background to simulate deep space. Collected NASA reference images for each planet — surface textures, color palettes, and relative size ratios — to guide the Python script's planet definitions.

Scene
02

Python Script — Planet Generation

Wrote a Python script using Blender's bpy API to procedurally generate all planets as UV spheres with correct relative scales (Jupiter ~11× Earth). The script automated object naming, placement along orbital distances, and Saturn's ring torus — eliminating repetitive manual work.

Scripting
03

UV Unwrapping & Texture Mapping

The script applied spherical UV unwraps to each planet programmatically. NASA diffuse/albedo texture maps were loaded via bpy.ops.image.open() and assigned through Image Texture nodes in the Shader Editor, with Mapping nodes to correct polar stretching.

Texturing
04

Materials & Shader Setup

Planet materials used Principled BSDF shaders, configured in the script via node trees. The Sun received a custom Emission shader with a color ramp for solar glow. Saturn's rings used a Transparent BSDF blended with Principled BSDF for semi-transparent opacity.

Shading
05

Lighting — The Sun as a Light Source

A Point Light was placed inside the Sun mesh via script with high intensity to cast realistic directional shadows across all planets. A low-strength ambient world light (0.05) was set to keep the dark sides of planets faintly visible without washing out the scene.

Lighting
06

Python Script — Orbital Animation

The script automated keyframe insertion for each planet's axial rotation and orbital motion using bpy.ops.anim.keyframe_insert(). Orbital speeds were derived from real planetary periods, scaled down proportionally. The camera was constrained to a Bezier curve path for a smooth orbit fly-through.

Animation
07

Star Field Background

A procedural star field was generated using Blender's World shader with a Noise Texture driving an Emission background — set up in the script via node group assignments. This avoided the overhead of a particle system while producing a convincing deep-space environment.

Environment
08

Render Settings & GPU Configuration

Render engine set to Cycles with GPU Compute enabled for the RTX 3050 Ti (CUDA) — configured programmatically via bpy.context.scene.cycles. Samples set to 256 for stills and 128 for animation frames, with OptiX denoising enabled to reduce noise without increasing sample count.

Render Setup
09

Post-Processing in Blender Compositor

Used Blender 5.1's Compositor to add a Glare node (Fog Glow) around the Sun and bright stars for a bloom effect. Color Balance node applied a cinematic grade — cooled shadows and warmed midtones. Node layout was partially automated through the Python script.

Compositing
10

Final Export

Rendered as a PNG image sequence (1920×1080) and compiled into MP4 using Blender's Video Sequence Editor with FFmpeg / H.264 encoding at 24 fps. The script set output paths and format settings automatically before the render batch was launched.

Export

Project Notes

Project goal: Recreate a visually accurate Solar System in 3D using Blender 5.1, automating the scene construction with a custom Python script via the bpy API — covering scripted modelling, texturing, lighting, orbital animation, and GPU rendering.

Challenges: Managing extreme scale differences between planets while keeping all objects visible in one scene required creative proportional scaling in the script. Achieving correct orbital motion via programmatic keyframing — and getting Saturn's ring transparency right through node tree scripting — took the most iteration.

Learning outcomes: Gained hands-on experience with Blender 5.1's Python API (bpy) for procedural scene generation, shader node automation, and render configuration. The project covered the full pipeline from scripted mesh creation through Cycles GPU rendering with OptiX denoising, compositing, and final video export.

Time spent: Approximately 12–15 hours across script development, texturing, lighting tweaks, and rendering sessions.

The Sun

Star · G-type

The Sun is the star at the center of our Solar System — a nearly perfect sphere of hot plasma that accounts for 99.86% of the total mass of the Solar System. Its core reaches temperatures of 15 million °C, fusing hydrogen into helium and releasing the energy that powers all life on Earth.

Surface Temp ~5,500°C
Core Temp ~15 million °C
Diameter 1.39 million km
Mass 1.989 × 10³⁰ kg
Age ~4.6 billion yrs
Distance (Earth) 149.6 million km
🌙

The Moon

Natural Satellite

Earth's only natural satellite and the fifth-largest moon in the Solar System. It stabilizes Earth's axial tilt, drives ocean tides, and is the only extraterrestrial body humans have set foot on — first reached during Apollo 11 in 1969.

Surface Temp -173 to 127°C
Diameter 3,474 km
Distance (Earth) 384,400 km
Orbital Period 27.3 Earth days
Gravity 0.165g
First Landing Apollo 11 (1969)
